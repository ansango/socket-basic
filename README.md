## Getting Started <a name = "getting_started"></a>

```bash
git clone
```

### Prerequisites

- Node / Npm
- Yarn

### Installing

```bash
yarn install | npm install
```

### Development

```bash
yarn dev | npm run dev
```

### Production

```bash
yarn start | npm start
```
